How Map Colors Are Calculated

When All Partners and All Themes are selected, the map color for each country is determined by computing a weighted average score across all its data rows:

Status Points
In Place2
Partially in Place1
Not in Place0

The average of all scores is then mapped to a final color:

Average ≥ 1.25 → Green (mostly in place)
Average ≥ 0.5 → Yellow (mixed / partial)
Average < 0.5 → Red (mostly not in place)

Example — Austria (all filters active):
Austria has 13 rows: 9 green, 0 yellow, 4 red.

score = (9 × 2 + 0 × 1 + 4 × 0) / 13
score = 18 / 13 = 1.38
1.38 ≥ 1.25 → Green

When a Partner or Theme filter is active, only the matching rows are used in the calculation — so the map updates dynamically to reflect the filtered selection.

When both a Partner and a Theme are selected and only one row matches per country, the color maps directly: green → green, yellow → yellow, red → red.
